¿Qué necesito para usar la app?
Luego de descargar la aplicación, deberás introducir tus datos personales, así como tu documento de identidad. Al terminar este paso, deberás validar el correo electrónico que recibirás y registrar tu medio de pago. Tras corroborar datos, ya estás listo para usar Bruno.
¿Por qué es más seguro realizar tus compras por la app?
El medio de pago se realiza en la pasalera de pagos de Visa y Mastercard. El código de barras se emplea para identificar al cliente, pero no contiene información del usuario, contraseña, financiera o el PIN. Para mantener tu seguridad, la aplicación te solicita reintroducir un PIN de cuatro dígitos cada vez que quieras realizar un pago de combustible
¿En qué dispositivos puedo usar la app?
Bruno está disponible para dispositivos iPhone (iOS versión 111 o superior) y Android (versión 5.0 o superior)
